How To Play

🪵 Build Along Edges

Read the task details in the top bar. Instead of picking boxes, you are building fences along the edge segments between the plots.

📐 Track Total Perimeter

Tap any grey line on the workspace board to set up an active wooden fence segment. Each segment counts as exactly 1 centimetre of boundary length.

❤️ Secure the Garden

Add up the segments to match the requested perimeter limit precisely. Tap the verification button to verify. You have two hearts per level before the lines fill in automatically.
